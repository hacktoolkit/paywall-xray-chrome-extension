# Paywall X-ray Chrome Extension

This plugin equips your browser with X-ray vision to see the content you've already downloaded behind paywall popups on various publications.

## Development and Testing

1. Clone this repository locally.
1. In Google Chrome, go to `Manage Extensions` (<chrome://extensions/>).
1. Enable `Developer mode`.
1. Press `Load unpacked` and navigate to the folder with `manifest.json`, and `Select`.
1. Voil√†!
